##  CHAPTER 03: 변수와 자료형 

### 문자형
- 문자를 입출력 하는 프로그램
```C
#include<stdio.h>

int main(void)
{
  char c; //변수 선언

  printf("문자를 입력하시오: ");
  scanf("%c", &c);

  printf("입력된 문자는 %c입니다\n", c);
  return 0;
}
```
