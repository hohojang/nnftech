##  CHAPTER 07: 배열

- 배열
  배열을 사용하면 한 번에 여러 개의 변수를 생성할 수 있음.
  10개의 정수를 저장할 수 있는 배열은 다음과 같이 선언함.
```c
int s[10];
```
  배열 안에 들어있는 각각의 데이터들은 번호에 의해 식별됨.

- 배열의 초기화
  배열을 초기화하려면 값들의 리스트를 콤마로 분리하여 중괄호{}로 감싼 후 배열을 선언할때 대입하면 됨.
```c
int score[5] = { 10, 20, 30, 40, 50};
```

##  CHAPTER 08: 함수

- 함수
  함수란 특정 작업을 수행하여 그 결과를 반환하는 문장들의 집합임.
  함수는 입력 데이터를 받아서 처리한 후에 결괏값을 반환함.
  각각의 함수에는 이름이 붙어 있으며 이름을 사용하여 함수를 호출하여 사용할 수 있음.

- 반환형
  함수가 처리를 종료한 후에 호출한 곳으로 반환하는 데이터의 유형을 말함.

- 매개변수
  함수가 받는 데이터를 함수 몸체로 전달해주는 변수임.
  각 매개 변수는 쉼표로 분리됨. add() 함수의 매개 변수는 int x와 int y임.

- 함수 몸체
  함수 몸체는 중괄호 {}로 둘러싸여 있음.
  함수가 호출되면 처음부터 순차적으로 하나씩 실행되며 return 문장을 만나게 되면 함수의 실행이 종료되고
  호출한 곳으로 되돌아감.
```c
int get_integer()
{
  int value;
  printf("정수를 입력하시오 : ");
  scanf("%d", &value);
  return value;
}
```
- 함수 호출
  함수를 사용하기 위해서는 함수를 호출해야함. 함수를 호출하려면 함수의 이름을 써주고 함수가 필요로 하는
  데이터를 나열한 다음, 세미콜론을 붙이면 됨.
```c
n = get_integer();
```

- 인수와 매개 변수
  인수(argument)는 호출 프로그램에 의해 함수에 전달되는 값임. 함수가 호출될 때마다 인수는 함수의 매개
  변수로 전달됨.
  함수가 호출될 때마다 인수는 달라질 수 있음. 매개 변수의 개수와 인자의 개수는 정확히 일치해야 함.
- 반환 값
  호출된 함수가 호출된 곳으로 작업의 결괏값을 전달하는 방법임.
  반환되는 값이 없다면 return 키워드 다음 아무것도 쓰지 않으면 됨.
  함수가 값을 반환하지 않는다면 반환형은 void로 정의해야 함.
- 함수 원형
  일반적으로 함수를 사용할 때 미리 컴파일러에게 함수에 대한 정보를 알려야 함.
  함수 원형은 함수가 사용되기 전에 정의되어야 함
```c
#include<stdio.h>
int compute_sum(int n);

int main(void)
{
}
```
- 지역 변수와 전역 변수
  함수 안에서 정의되는 변수는 지역 변수라고 불리고 해당 함수 안에서만 사용이 가능함.
  함수의 외부에서 선언되는 변수는 전역 변수라고 불림. 또한 전역 변수는 소스 파일 어디서나 사용 가능.

- 지역 변수
```c
#include<stdio.h>
int compute_sum(int n);

int main(void)
{
  int sum; ///< 지역 변수
  sum = compute_sum(100);
  printf("1부터 100까지의 합=%d \n", sum);
retrun 0;
}
  int compute_sum(int n)
{
  int i;
  int result = 0;
  for (i = 1; i <= n; i++)
      result += i;
  return result;

```
- 전역 변수
  함수 외부에서 선언되는 변수임. 전역 변수의 범위는 소스 파일 전체이며 즉 전역 변수는 소스 파일 안의 모든
  함수에서 사용이 가능한 변수.
```c
#include<stdio.h>

int global = 123; ///< 전역 변수

void sub1()
{
  printf("%d\n", global);
}

void sub2()
{
  printf("%d\n", global);
}

int main(void)
{
  sub1();
  sub2();
  return 0;
}
```
- C언어에서의 변수의 범위(Scope)
- 지역변수
  함수 또는 블록{} 안에서 선언된 변수
  선언된 블록 안에서만 사용 가능
  함수가 끝나면 메모리에서 사라짐
```c
void func(){
  int x = 10; // x는 func 안에서만 사용 가능
}
```
- 전역변수
  함수 밖에서 선언된 변수
  모든 함수에서 접근 가능
  프로그램이 시작될 때 생성되고, 끝날 때까지 유지됨
```c
int g = 100;

void show() {
  printf("%d", g); // 어디서든 접근 가능
}
``` 
- 매개변수
  함수의 매개변수로 전달된 변수
  함수 내부에서만 사용 가능 (지역 변수처럼 작동함)
```c
void print(int n){ // n은 매개변수 범위
  print("%d", n);
}
```
- 정적 변수
  지역 변수처럼 블록에서만 사용되지만 블록을 벗어나도 자동으로 삭제되지 않으려면 지역 변수를 정적 변수(static variable)로 만들면 가능.
  키워드 static을 앞에 붙임. 정적 변수는 전역 변수와 같이 프로그램이 시작할 때 생성되고 프로그램이 실행을 종료하면 삭제됨.

- 순환 호출
  함수는 자기 자신을 호출할 수도 있음. 이것을 순환(recursion)라고 부름.

##  CHAPTER 09: 포인터

- 포인터
  메모리의 주소를 가지고 있는 변수임. 포인터는 변수가 저장되는 주소와 깊은 관계가 있음.
  변수는 자료들을 저장하는 역할을 하므로 메모리에 만들어짐.
  메모리는 바이트로 구성되고 각 바이트마다 고유한 주소를 가지고 있음.
  프로그램에서 변수를 만들면 이들 변수는 컴파일러에 의해 메모리 공간에 배치됨.
  일반적인 PC 환경에서 char형 변수는 1바이트, int형 변수는 4바이트, float형 변수는 4바이트를 차지함
- 정수를 가리키는 포인터
```c
int *p;
```
p는 정수를가리키는 포인터임. 포인터도 변수이기에 포인터를 사용하기 전에 선언되어야 함.
포인터를 선언하려면 포인터가 가리키는 자료형을 쓰고 *을 붙인 다음, 변수의 이름을 씀. 
포인터는  사용하기 전에 반드시 초기화를 하여야 함. 
포인터에는 변수의 주소가 저장되어야 하므로 & 연산자를 이용하여 주소를 계산하여 포인터에 대입하여 주면 됨.

```c
int number = 10;  // 정수형 변수 number 선언
int *p;           // 포인터 p의 선언
p = &number;      // 변수 number의 주소가 포인터 p로 대입 
```

- 간접 참조 연산자 *
  포인터가 단순히 메모리 주소만 저장할 수 있는 것이라면 별로 유용하지 않을것임.
  포인터를 통하여 포인터가 가리키는 위치의 값을 읽어오거나 변경할 수 있음.
  포인터 p가 가리키는 위치에 저장된 내용을 가져오려면 p 앞에 * 기호를 붙여 *p하면 됨.
  이것을 포인터를 통하여 간접 참조한다고 함.
```c
int number = 10;
int *p;
p = &number;

printf("%d\n", *p);  // 10이 출력 됨
```
- 포인터 연산
  포인터에 대해서는 덧셈과 뺄셈 연산만 가능함. 포인터 변수에 대한 연산은 일반적인 변수에 대한 연산과는 조금 다름.
  포인터에 증가 연산인 ++을 적용했을 경우, 증가되는 값은 포인터가 가리키는 객체의 크기임.
  따라서 char형 포인터를 증가시키면 char형의 크기인 1바이트만큼 증가함.

- swap() 함수 작성하기
```c
swap(&a, &b);
```
함수 호출 시 인수가 a, b가 아닌 &a, &b인 것에 유의해야 함.
즉 변수의 값을 전달하는 것이 아니라 변수의 주소를 전달함.
주소를 받을 수 있는 자료형은 포인터임. 따라서 swap()은 다음과 같이 정의되어야함.
```c
void swap(int *px, int *py)
```
- 포인터 사용시 주의할 점
  초기화하지 않고 사용하기
  만약 포인터가 선언만 되고 초기화 되지 않았다면 포인터는 임의의 주소를 가리키게 됨. 이러한 상태에서 포인터를 이용하여 메모리의 내용을
  변경한다면 문제가 발생할 수 있음.

  널 포인터의 사용
  포인터가 아무것도 가리키고 있지 않을 때는 NULL(0)으로 설정하는 것이 바람직함.
```C
int *p = NULL;
```
- 배열과 포인터
  배열과 포인터는 아주 밀접한 관계를 가지고 있음. 배열 이름 그 자체가 포인터임.

##  CHAPTER 10: 문자열
- 문자열
  문자열(string)은 "Hello"와 같이 문자들의 모임임. 문자열은 큰 따옴표를 이용해 표현됨.
```c
"A"
"Hello"
"Mr. Hong"
```
  문자열은 어디에 저장하는가? 문자형 변수는 하나의 문자만을 저장할 수 있음. 
  예를 들어, 문자열 "Hello"를 저장하려면 다음과 같이 문자형 배열을 선언하고 초기화함.
```c
char str[6] = "Hello";
```
  각각의 문자는 배열 요소에 저장됨. 위 코드 "Hello"에는 5개의 문자만 있는데 왜 배열의 크기는 6인지?
  C에서는 문자열의 끝이 반드시 NULL 문자라는 특수한 값으로 끝나야하기 때문임. 따라서 문자열을 저장하는 문자 배열은 문자열의 크기보다 
  항상 하나 더 커야 함.
  
- 문자형 배열의 초기화
  문자형 배열을 초기화 하는 두가지 방법
  첫 번째 방법은 배열을 초기화 하듯 각 배열 요소 값들을 중괄호 안에 넣어주는 방법임. 이 경우 배열의 마지막 요소에는 반드시 NULL문자 넣기.
```C
char str[6] = {'H','e','l','l','o','\0'};
```
  두 번째 방법은 문자열 상수를 이용하여 초기화 하는 방법.
  배열의 요소에는 문자열 상수 안의 문자들이 자동으로 저장됨. 이 방법을 사용하면 컴파일러가 자동으로 문자열의 끝에 NULL 문자를 추가함.
```C
CHAR STR[6] = "Hello";
```
  만약 배열의 크기를 지정하지 않으면 컴파일러가 문자의 개수에 맞추어 자동으로 배열의 크기를 설정함
```c
char str[] = "Hello";
```
- 문자열 입출력
  문자열을 입력하거나 출력하려면 다음과 같은 함수들을 사용할 수 있음.
```c
scanf("%s",s)  // 문자열을 읽어서 배열 s[]에 저장
printf("%s",s) // 배열 s[]에 저장되어 있는 문자열을 출력한다.
gets_s(char s[], int length) // 한 줄의 문자열을 읽어서 배열 s[]에 저장한다.
puts(char s[]) // 배열 s[]에 저장되어 있는 한 줄의 문자열을 출력한다.
```
